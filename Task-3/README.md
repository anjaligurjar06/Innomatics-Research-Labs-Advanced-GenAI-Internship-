# Python Programming Assignment  
## Data Processing and Analysis

###  Introduction
This assignment focuses on applying Python programming concepts to real-world scenarios involving data processing, validation, and analysis.  

The programs use basic Python data structures like:
- Lists  
- Dictionaries  
- Strings  
- Tuples  

Each problem demonstrates simple logic building using loops and conditions.

##  Problem Statements

### 1️⃣ Employee Performance Bonus Eligibility
- Find the highest performance score.
- Handle ties if multiple employees have the same score.
- Display all employees eligible for the top bonus.

---

### 2️⃣ Search Query Keyword Analysis
- Convert input to lowercase.
- Remove common punctuation.
- Count frequency of each keyword.
- Display only keywords appearing more than once.

---

### 3️⃣ Sensor Data Validation
- Identify even sensor readings.
- Store them as (hour_index, reading_value).
- Ignore odd readings.

---

### 4️⃣ Email Domain Usage Analysis
- Count number of users per email domain.
- Calculate percentage usage of each domain.

---

### 5️⃣ Sales Spike Detection
- Calculate average daily sales.
- Detect days where sales are more than 30% above average.
- Display day number and sale value.

---

### 6️⃣ Duplicate User ID Detection
- Identify duplicate user IDs.
- Display how many times each duplicate appears.
